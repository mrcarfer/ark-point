# ARK Point

Sistema inteligente de controle de jornada para equipes remotas.

## Objetivo

Criar um sistema capaz de:

- registrar ponto online
- controlar jornada
- gerar relatórios
- monitorar produtividade
- gerar alertas inteligentes
- servir como base do ecossistema ARK-T

## Stack planejada

Frontend:
- React
- TypeScript
- Tailwind

Backend:
- NestJS
- Prisma
- PostgreSQL
